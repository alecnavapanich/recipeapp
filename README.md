## Getting Started / Dependencies

* Install [Node](https://nodejs.org/en) LTS and [Java SE Development Kit (JDK)](https://openjdk.org/projects/jdk/17/)
    * If you have [Chocolatey](https://chocolatey.org/install), you can install them with this command:
`choco install -y nodejs-lts microsoft-openjdk17`
* Download the Expo GO App on your [Android](https://play.google.com/store/apps/details?id=host.exp.exponent) or [Apple](https://apps.apple.com/us/app/expo-go/id982107779) device for fast and easy testing
